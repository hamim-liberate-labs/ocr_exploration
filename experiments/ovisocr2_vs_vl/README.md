# OvisOCR2 vs PaddleOCR-VL 1.6 — Modal L4 run + Gemini LLM-judge

A different track from run1–run4. Those tested light **PaddleOCR** configs (PP-OCRv6 / PP-StructureV3)
on a Kaggle T4 with `notebooks/ocr_vl16_comparison.ipynb`. This run tests **OvisOCR2** — a compact
~0.8B document-parsing VLM (Qwen-based) that emits Markdown + HTML tables + LaTeX in one pass — as a
lightweight stand-in for **PaddleOCR-VL 1.6**, and adds an **LLM-as-judge** quality verdict that the
VL-as-pseudo-reference metrics can't give.

Everything is in the self-contained **`notebook.ipynb`** (run top-to-bottom).

## What it does

1. **Runs OvisOCR2 on the 100 `test_dataset_100` images on a Modal L4 GPU** (vLLM) → per-page `.md`,
   `.html`, and a manifest (`latency, chars, n_tables`). The Modal app is the sidecar
   **`ovisocr2_modal_app.py`** (must stay next to the notebook).
2. **Loads the existing VL 1.6 outputs** from `../100img_run4_sv3_ocrv6_lean/results/vl_1_6/` as the
   pseudo-reference (no new VL run needed).
3. Three evaluation layers:
   - **Layer 3 — divergence vs VL** (word P/R/F1, char-sim, table recall) — the *same* metric
     functions as run1–run4, so numbers are comparable. Measures divergence, not who's better.
   - **Layer 1 — operational + robustness** — latency, cost/page, and failure flags
     (empty / degenerate repetition / unbalanced `<table>` HTML), completeness (chars, tables).
   - **Layer 2 — quality verdict (the winner call)** — **Gemini 2.5 Flash** LLM-judge sees the
     source image + both transcriptions and picks the more faithful one (text fidelity, table
     structure, reading order), with position-bias randomized per page.
4. **Results** — win-rate overall + per category, mean sub-scores, figures, and a recommendation.

## Run it

- **Modal:** `modal token new` once (GPU run). **Gemini:** `GOOGLE_API_KEY` in the repo-root `.env`
  (loaded automatically by the notebook).
- Open `notebook.ipynb` → **Run All**. `MAX_IMAGES = None` (in §1) runs the full 100; set an int for
  a smoke test.
- Cost: OvisOCR2 run = Modal L4 GPU minutes; Gemini judge ≈ $0.10–0.15 for 100 pages.

## Why OvisOCR2

run1–run4 concluded that **no light PaddleOCR model fully replaces VL 1.6**: PP-OCRv6 captures VL's
words but not reading order; PP-StructureV3 captures tables but still misses ~1/3 of VL's text.
OvisOCR2 is a small VLM that produces text **and** tables **and** layout in a single pass — this run
measures whether it closes that gap, judged directly against the page (LLM-judge) rather than only
by agreement with VL.
